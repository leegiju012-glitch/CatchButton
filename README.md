--4단계코드의 실행 스크린샷
초기 버튼 상태와 난수 재사용을 위해 class 창을 수정하였고 misscount가 20회가 된다면 모든 버튼을 사용 안되게 함.
그리고 바로 게임오버 메세지창을 생성하도록 함.
다시시작 버튼을 누르면 점수, 놓친 횟수, 버튼 크기, 위치를 초기화하고 버튼을 다시 활성화하도록 함.
